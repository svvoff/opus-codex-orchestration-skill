# Roadmap Stages

- Stage 0 — Discovery / concept validation
- Stage 1 — MVP
- Stage 2 — Validation / soft launch
- Stage 3 — Public launch
- Stage 4 — Retention / engagement
- Stage 5 — Monetization expansion
- Stage 6 — Scale / live operations

Roadmap stages are not fixed dates unless the user explicitly needs date-based planning.
