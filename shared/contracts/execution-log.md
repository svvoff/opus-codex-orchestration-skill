# Contract: Execution Log

Recommended paths:

```text
docs/ai/execution-log/latest.md
docs/ai/execution-log/archive/YYYY-MM.md
```

## latest.md

Contains only recent completions, usually the last 5–10 tasks.

## Archive

Monthly summaries only. Full reports should live with done task cards only when needed.

## Entry fields

- Task ID
- Date
- Result
- Opus decision
- Validation evidence summary
- Changed areas
- Follow-up
