# Contract: Configurator Input Packet

A compact bridge from product discovery to Opus/Codex configuration.

Path in target repo:

```text
docs/product/configurator-input.md
```

## Required sections

- Project summary
- Current stage
- Likely technical shape
- Product constraints that affect engineering
- Roadmap-aware guardrails
- Risk areas for Opus/Codex config
- Validation priorities
- Recommended task style
- Known non-goals
- Open technical questions

## Rule

This file should be short enough to read every time the configurator runs. It should not duplicate the full roadmap or backlog.
