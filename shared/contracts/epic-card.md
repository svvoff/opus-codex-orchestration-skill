# Contract: Epic Card

Recommended paths:

```text
docs/product/backlog/epics/active/EPIC-001-name.md
docs/product/backlog/epics/done/EPIC-001-name.md
```

## Required sections

- ID
- Title
- Status
- Stage
- Goal
- Success criteria
- Scope
- Non-goals
- Task list
- Risk areas
- Validation expectations
- Exit criteria

## Token-aware rule

Active epics should be concise. Done epics should be summarized and moved to `epics/done/`.
