# Contract: Backlog Index

Path:

```text
docs/product/backlog/index.md
```

## Purpose

Route Opus to the next small piece of context without reading the entire backlog.

## Required sections

- Active milestone
- Next task selection policy
- Ready tasks table
- Blocked tasks summary
- Recently completed summary
- Do-not-read-by-default paths

## Ready tasks table fields

- ID
- Title
- Epic
- Priority
- Risk
- Depends on
- File

## Rule

The index should not contain full task bodies. It should point to task card files.
