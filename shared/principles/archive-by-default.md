# Archive By Default

Completed, deferred, obsolete, or historical work must be placed outside the active context path.

## Archive folders

Recommended target-project folders:

```text
docs/product/backlog/tasks/done/
docs/product/backlog/tasks/deferred/
docs/product/backlog/epics/done/
docs/product/decisions/archive/
docs/ai/execution-log/archive/
```

## Reading policy

Archives are not read during normal execution. They are read only for:

- dependency verification;
- regression investigation;
- user-requested history;
- release notes;
- reconstructing why a decision was made.
