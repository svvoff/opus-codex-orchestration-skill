# Fallback Orchestrator Codex Skill

Use this repo-scoped Codex skill when Opus is unavailable or when the user intentionally wants to save Opus tokens on simple work.

Example calls:

```text
$fallback-orchestrator обсудим новый проект
$fallback-orchestrator настрой связку для ../target-repo
$fallback-orchestrator делай следующую простую задачу
```

This skill is conservative. It is not a full replacement for Opus on strategy, architecture, high-risk review, or validation policy design.
