# EPIC-001 — Core Gameplay Loop

## Goal

Create the smallest playable version of the game.

## Success criteria

- Player can start, play, lose, and restart.
- Score is visible.
- Game runs in browser.

## Non-goals

- Ads
- Leaderboard
- Backend
- Advanced analytics
